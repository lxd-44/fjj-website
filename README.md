# Francis Jiu Jitsu Website

Official website for Francis Jiu Jitsu - Perth's most welcoming BJJ academy.

## Tech Stack
- HTML5 + Tailwind CSS (via CDN)
- Vanilla JavaScript
- Hosted on Netlify

## Pages
- `index.html` - Homepage
- `classes.html` - Class offerings
- `schedule.html` - Timetable with booking links
- `about.html` - About the academy
- `faq.html` - Frequently asked questions
- `contact.html` - Contact form

## Design System
- **Display Font**: Outfit (900 weight)
- **Body Font**: DM Sans
- **Primary Colour**: Ember (#EC6433)
- **Background**: Paper (#FAF8F5) / Ink (#1A1A1A)
- **Style**: Zen Brutalist + Glassmorphism

## Deployment
Automatically deploys to Netlify on push to `main` branch.

## Booking Integration
All booking links point to TidyCal: https://tidycal.com/fjj
